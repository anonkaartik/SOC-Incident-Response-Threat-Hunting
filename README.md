# SOC Incident Response & Threat Hunting Platform

## Overview

This project simulates a Security Operations Center (SOC) investigation.

The platform analyzes security logs, detects threats, identifies Indicators of Compromise (IOCs), and generates incident response findings.

## Features

- Log Analysis
- Threat Detection
- IOC Extraction
- Incident Response
- Threat Hunting
